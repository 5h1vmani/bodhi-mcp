# D2C Community Building

> Platform-specific decision framework for building brand communities that drive retention and reduce CAC.

## TL;DR

- **Platform choice = community type** — Reddit for product feedback loops, Facebook Groups for peer support, X for brand personality, Instagram for visual co-creation and inner-circle exclusivity
- **Engagement ratios differ by platform** — Reddit 90/10, X 70/30, Facebook Groups 50/30/20, Instagram Reels-heavy (74% shift from static) with Broadcast Channels for bypass-the-algorithm direct reach
- **Negativity is platform-shaped** — Reddit punishes inauthenticity (disclose always), X rewards speed (reply > clever posts), Facebook Groups need upfront rules (prevent > react)
- **Co-creation > broadcasting** — Glossier's 300+ comment product dev posts, Gymshark's #Gymshark66, Peloton's instructor-influencer model all turned customers into builders
- **Community-driven brands see 40% higher retention** and 150% higher LTV (Three Ships referral community) vs non-community channels

## Decision Guide

| Scenario                                       | Platform + Approach                                                      | Why                                                                                                  |
| ---------------------------------------------- | ------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| Need product feedback loop                     | Reddit: dedicated subreddit + dual accounts (brand + named rep)          | Tech-savvy users give blunt, detailed feedback; 90/10 rule builds trust                              |
| Building habit/lifestyle brand                 | Facebook Group: private, questionnaire-gated, themed weekly threads      | Peer-to-peer support drives 3x engagement vs pages; algorithm prioritizes group content              |
| Brand personality is a differentiator          | X: distinctive voice + 70/30 engage-others ratio                         | Replies are remembered more than posts; personality compounds (Wendy's, Dollar Shave Club)           |
| High-ticket product, trust-critical            | Reddit AMA + Facebook Group exclusivity                                  | AMA builds founder credibility; exclusive group creates belonging for committed buyers               |
| Launching challenge/movement                   | X + cross-platform hashtag (#Gymshark66 model)                           | X amplifies viral loops; challenge format drives participation over passive consumption              |
| Scaling beyond founder-led community           | Facebook Group with volunteer moderators (Elementor model, 115K members) | Facilitate-don't-dominate; multiple admins ensure response speed without founder bottleneck          |
| Crowd-sourcing product decisions               | Reddit product dev posts (Glossier model)                                | 300+ detailed comments per post; community feels ownership → 600% sales growth                       |
| Retention/reducing churn                       | Facebook Group with member-only perks + gamification (Peloton model)     | Exclusivity drives 2x product recommendations; leaderboards create switching costs                   |
| Visual/lifestyle product, younger audience     | Instagram: Broadcast Channel for mega-fans + Close Friends for VIP drops | Broadcast Channels bypass algorithm (direct to inbox); 74% content shift to Reels = discovery engine |
| Rapid product feedback on aesthetics/packaging | Instagram: Stories polls + question stickers → product decisions         | Low-friction input (tap to vote); Glossier used this to co-create packaging with 800K followers      |
| Building inner circle / superfan tier          | Instagram: Close Friends stories + Broadcast Channel exclusives          | Tiered access creates status; early drops and behind-the-scenes reward loyalty without gating        |

## Community Type Selection (McKinsey Framework)

| Type           | Definition                            | Best For                                     | D2C Example                               |
| -------------- | ------------------------------------- | -------------------------------------------- | ----------------------------------------- |
| Directive      | Brand sets agenda, members follow     | Product launches, announcements              | Apple product reveals                     |
| Transactional  | Exchange-based (discounts, referrals) | Quick growth, low-margin products            | Three Ships "The Fleet" (150% higher LTV) |
| Conversational | Peer discussion, brand facilitates    | High-engagement categories (fitness, beauty) | Instant Pot (3.1M members, recipe-driven) |
| Experiential   | Shared activities, challenges, events | Lifestyle/identity brands                    | Gymshark meetups, Peloton leaderboards    |

## Platform Engagement Ratios

| Platform        | Ratio                                             | What It Means                                                                                                                                     |
| --------------- | ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Reddit          | 90/10 (value / brand content)                     | 90% helpful comments with zero promotion; 10% brand-related. Anything more triggers "shill" detection                                             |
| X (Twitter)     | 70/30 (engage others / original posts)            | 70% replying, retweeting, engaging; 30% original content. Native media performs 40% better than links                                             |
| Facebook Groups | 50/30/20 (value / storytelling / promotion)       | 50% educational/helpful, 30% brand narrative, 20% direct promo. Daily check-ins > daily posts                                                     |
| Instagram       | Reels-first (74% of content) + Broadcast Channels | Feed for discovery (Reels, Collabs), Stories for engagement (polls, Q&A), Broadcast Channel for direct community access without algorithm penalty |

## Handling Negativity

| Platform        | Norm                                  | Approach                                                                                                                                                                                           |
| --------------- | ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Reddit          | Radical transparency expected         | Always disclose brand affiliation. Treat criticism as product feedback. Never suppress or astroturf — users detect and punish fast                                                                 |
| X               | Speed and personality expected        | Reply to all mentions promptly. Treat responses as relationship-building. Direct engagement with concerns (Warby Parker model)                                                                     |
| Facebook Groups | Prevention over reaction              | Establish 5-10 clear rules upfront (anti-bullying, privacy, spam). Questionnaire-gate membership. Multiple admins for swift response                                                               |
| Instagram       | Visibility management + DM resolution | Negative comments are public and persistent. Reply publicly with empathy, move to DM for resolution. Pin positive UGC comments. Hide (don't delete) abusive comments — deletion triggers reposting |

## Common Mistakes

| Mistake                                                       | Fix                                                                                                                   |
| ------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Same voice/strategy across all platforms                      | Adapt per platform norms: cheeky on X, helpful on Reddit, facilitative in Groups                                      |
| Undisclosed brand promotion on Reddit ("shilling")            | Always disclose. Use official brand account + named representative accounts                                           |
| Brand-dominated Facebook Group (every post is brand content)  | Facilitate peer conversations; respond when tagged, don't initiate every thread                                       |
| Treating community as marketing channel                       | Flip the ratio: community value first, promotion last. Co-create with members                                         |
| Building community after product-market fit                   | Start community pre-launch (Glossier's "Into the Gloss" blog preceded the brand)                                      |
| No forgiveness in gamified communities                        | Add freeze options for streaks, segment leaderboards, make participation optional                                     |
| Using Instagram feed as community hub (algorithm kills reach) | Use Broadcast Channels for direct reach + Close Friends for VIP tier; feed is for discovery, not community            |
| Ignoring Reels for community (treating IG like 2019)          | 74% content shift to Reels; static posts get fraction of reach. Reels for discovery → Broadcast Channel for retention |

## Checklist

- [ ] Platform selected based on community type needed (not "where competitors are")
- [ ] Engagement ratio defined per platform (90/10, 70/30, or 50/30/20)
- [ ] Negativity handling playbook documented per platform
- [ ] Community guidelines published before launch (Facebook Groups: 5-10 rules)
- [ ] Dual account strategy for Reddit (brand + named representative)
- [ ] Content calendar includes co-creation touchpoints (not just broadcast)
- [ ] Membership gating in place for exclusive communities (questionnaire for Groups)
- [ ] Instagram community uses Broadcast Channel (not just feed) for direct member access
- [ ] Success metrics defined: retention rate, LTV, referral rate (not vanity metrics)
